# Bankr Controls

Start with these controls.

## Wallet

Pause transfers above the user's small-spend limit.

Deny transfers above the hard limit.

Pause unknown recipients.

Deny unsupported chains.

## Trading

Pause swaps above the user's configured USD limit.

Pause high-slippage trades.

Deny assets on a blocklist.

## Token Launches

Pause every token launch.

Deny launches with missing metadata.

Pause launches that create external links or browser actions.

## Automations

Pause new DCA, stop-loss, leverage, or recurring actions.

Deny automations without a max spend or end condition.

## External Actions

Pause unknown domains.

Deny requests that include secret-looking values.

Deny browser actions against phishing-like domains.
